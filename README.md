## VibeRoute — веб-приложение на Flask


### 1. Требования

- Python 3.10+
- Интернет (для загрузки библиотек, а при необходимости — для геокодеров)
- Установленный `pip`

---

### 2. Создать и активировать виртуальное окружение (Windows)

В корне проекта (там, где лежит `main.py` и `requirements.txt`):

```
python -m venv venv

venv\Scripts\activate
```
(В терминале должно появиться `(venv)` в начале строки.)

---

### 3. Установка зависимостей
```
pip install -r requirements.txt
```

---

### 4. Запуск приложения

Из корня проекта:
```
python main.py
```
По умолчанию Flask поднимет сервер на:

- `http://127.0.0.1:5000/`  
  или  
- `http://localhost:5000/`

Открой этот адрес в браузере.
